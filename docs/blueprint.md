# **App Name**: Rupee Relief

## Core Features:

- Budget Input: Input section for total budget in rupees.
- Grocery Item List: A dynamic list of grocery items with name, price per unit, desired quantity, and priority.
- Add Item Button: Button to add more grocery rows.
- Calculate Button: Calculate button that runs the logic to allocate items within the budget based on priority and desired quantity.
- Allocation Results: Display list of items with final quantity allocated and total cost, sorted by allocation priority.
- Remaining Budget Display: Display remaining budget after allocation.

## Style Guidelines:

- Primary color: Light lavender (#E6E6FA) to evoke a sense of calm and organization. This hue is often associated with mindful activities, in contrast to the stress surrounding finances.
- Background color: Very light lavender (#F5F5FF), subtly different from the primary color.
- Accent color: Soft purple (#D8BFD8), a bit darker and more saturated than the primary color, to provide visual contrast on key UI elements without causing harshness.
- Body and headline font: 'Inter' (sans-serif) for a clean, accessible, and modern feel. This font provides high readability for both headings and body text.
- Responsive and mobile-friendly layout optimized with Tailwind CSS grid and flexbox utilities.
- Use soft shadows, rounded corners (2xl), and spacing (p-4, gap-4) to create a modern and visually appealing interface.
- Subtle animations using Framer Motion for item additions and results appearing to enhance user experience.
